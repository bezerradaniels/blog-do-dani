# 21st Documentation Summary

## Overview
21st is an AI-powered prototyping tool combined with a component library for AI-assisted UI building. It enables rapid prototyping and iteration on React applications.

## Core Features

### Canvas: AI-Powered Prototyping
- **Chat-based interface**: Describe what you want — from small tweaks to new features
- **Multiple prototypes**: Generate several variations simultaneously and compare them
- **Code extraction**: Copy your favorite prototype's code directly into your project
- **Real codebase integration**: Optionally connect GitHub to prototype with your existing components

### Community: Component Library
- **Installable components**: Real React code, not just screenshots
- **Multiple usage methods**:
  - Copy components and drop into your repo via Canvas
  - Use component links and prompts in Canvas chat
  - Paste prompts into AI coding assistants (Cursor, Claude Code, Lovable, etc.)

## Target Users
- **Frontend engineers** - Prototype and iterate on UI faster with AI coding agent
- **Designers** - Turn ideas, references, and screenshots into high-fidelity React components
- **Product teams & founders** - Prototype, compare approaches, and collect feedback before production code

## Key Differentiators
- **Multiple prototypes at once** - Generate several variations and compare visually
- **Works on your real codebase** - Connect GitHub to prototype with your components
- **Community of real components** - Installable, remixable React code
- **Context-aware AI** - Combines your repo, components library, and inspiration screens

## Canvas Features

### Core Modes
1. **Agent Mode** - Full-featured coding agent for direct edits
2. **Prototype Mode** - Generate multiple variations at once
3. **Design Mode** - Visual no-code editor for styling

### Chat & Commands
- **Inline @commands** for specific actions
- **Multiple chat modes** for different tasks
- **Context management** with on-canvas items, images, and @Codebase

### Integration Features
- **GitHub Integration** - Prototype with existing codebase
- **Context Management** - Add on-canvas items, images, and @Codebase references
- **Version History** - Versions dialog and restore checkpoints

### Code Export
- **Copy code** directly to clipboard
- **Export prompts** for use in other AI tools
- **Download complete prototypes**

## Community Features

### Component Library
- **Production-ready React/Tailwind components**
- **Real app UI screenshots** for inspiration
- **Copy color schemes and typography** from themes
- **Keyboard shortcuts** for faster navigation

## Getting Started

### Quick Start Flow
1. **Create Canvas** - Start with AI-powered prototyping
2. **Describe your needs** - Use chat to explain what you want
3. **Generate variations** - Get multiple prototype options
4. **Compare and select** - Choose the best approach
5. **Export code** - Copy to your project

### Integration Options
- **Standalone Canvas** - Use without codebase connection
- **GitHub-connected** - Prototype with your existing components
- **Community components** - Use pre-built React components
